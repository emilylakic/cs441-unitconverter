## CS441 UNIT CONVERTER ANDROID APP -- PROJECT 1

# What It Does

Converts a user-inputed number between the units meters, centimeters, and inches based on the user's choice. 

# Design Features

This app contains:
- 4x LinearLayout (3 horizontal, 1 vertical)
- 1x ImageView
- 1x EditText
- 4x TextView
- 2x Spinner
- 1x Button
